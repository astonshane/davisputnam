# "Classic" Style
- Literals
 - A
- Negation
 - NOT A
- Conjunctions
 - A AND B
- Disjunctions
 - A OR B
- Implication
 - A IMPLIES B
- Biconditional
    - A IFF B

#Sample Input file:
- NOTE: Last Input input line corresponds to the conclusion

 ```sh
A IMPLIES (N OR Q)
NOT(N OR NOT A)
A IMPLIES Q
```
